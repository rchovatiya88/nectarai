## 5. Model Selection Strategy

### Free-First Routing
1. Try free models first (Llama, DeepSeek, Qwen)
2. Quality gate: if output poor, retry with paid model
3. Human override for critical tasks (code, security)
4. Client can request premium (upsells automatically)

### Model Tiers
- **Free:** Llama 3.3 70B, DeepSeek V4 Flash, Qwen3 Coder
- **Quality ($3/M tokens):** Claude Sonnet 4.6, GPT-4o
- **Premium ($15/M tokens):** Claude Opus 4 (rarely needed)
